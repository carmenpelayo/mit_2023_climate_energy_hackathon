# *MIT 2023 Energy & Climate Hackathon* - Schneider Electric's Cold-Start Challenge
Schneider Electric's "Cold Start Challenge” prompted teams to find the optimal time-series forecasting model to predict the energy consumption of buildings for which we have little data, in order to maximize energy savings in the long run. We created a machine learning model to forecast the energy load in new buildings based on historical data of older ones. We used AutoML capabilities for time series (autoTS) and identified the ExtraTreeRegressor algorithm as the best-performing estimator, which we later trained over a> 1.4M-record dataset. The resulting model had an accuracy of 83% (R-squared score) on unseen data.
